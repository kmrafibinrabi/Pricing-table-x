== Description ==

Pricing Table X is a simple and customizable pricing table plugin for WordPress.
Easily create and manage pricing tables for your products or services.

= Key Features =

- **Easy to Use:** Intuitive and user-friendly interface.
- **Customization:** Customize the appearance to match your website's design using custom style.
- **Responsive:** Pricing tables are fully responsive and look great on all devices.
- **Multiple Columns:** Create pricing tables with multiple columns to showcase various plans.
- **Currency Options:** Choose from a variety of currency options to suit your audience.
- **Highlight Features:** Highlight key features of each pricing plan.
- **Link to Order:** Directly link each pricing plan to the order or signup page.
- **Regular Updates:** We are committed to providing regular updates and improvements.

== Manual Installation ==

1. Upload the entire `vx-pricing-table` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Pricing Tables menu in the admin area to create your pricing tables.

= Automatic installation =

Automatic installation is the easiest option. Log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
